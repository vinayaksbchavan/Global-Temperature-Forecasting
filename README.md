# 🌡️ Forecasting Global Temperature Anomalies

A machine learning project to forecast global surface temperature anomalies using 145 years of NASA climate data (1880–2024).

# 📌 Overview
Climate change is one of the most critical issues of our time. This project builds a predictive model to forecast global temperature anomalies and analyze long-term climate trends using historical NASA data.

# 🚀 Results
| Metric | Score |
|--------|-------|
| R² Score | 0.86 |
| RMSE | 0.15 |
| MAE | 0.10 |

# 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Model:** Gradient Boosting Regressor (tuned via GridSearchCV)

# 📊 What's Inside
- Exploratory Data Analysis (EDA) on 145 years of NASA temperature data
- Feature engineering — lag features and rolling window features
- Model training and hyperparameter tuning using GridSearchCV
- Model evaluation using R², RMSE, and MAE
- Long-term climate trend analysis and visualization

# 📁 Dataset
- Source: NASA GISS Surface Temperature Analysis
- File: `GLB_Ts_dSST.csv`
- Coverage: 1880–2024 (Global Land-Ocean Temperature Index)
